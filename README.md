# Polyrhythm
Recreating of an audio visual animation inspired by [project_jdm on youtube](https://www.youtube.com/@project_jdm)

## Dependencies:
- SFML 2.6

## How to install:
```
$ cargo install --git https://github.com/Pranjal-Patel/polyrhythm
$ polyrhythm
```
